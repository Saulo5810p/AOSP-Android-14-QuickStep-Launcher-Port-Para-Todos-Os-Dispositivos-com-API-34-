# Porta de lançamento de passo rápido do Android 14 AOSP

Este é um port funcional do Launcher padrão (Launcher3/QuickStep) do AOSP baseado no Android 14, modificado para funcionar em outros dispositivos e versões do Android.

##  📱 Características
*  Experiência pura do launcher stock do Android 14 (AOSP).
*  Correções de crashes e incompatibilidades aplicadas diretamente via Smali

Funções do Launcher3: O básico do básic9. Não oculta após, não remove labels, apenas rotaciona a tela, tem ponto de notificações, e é só isso mesmo. Bem seco. Porém tem seu charme, como o menu de aplicativos e a grade da tela inicial que muda de cor combinando com seu papel de parede.

##  🛠️ Como foi feito
O projeto foi modificado inteiramente em um dispositivo móvel utilizando o  **MT Manager**  para depuração, engenharia reversa e correção de bugs no código Smali.
